cMatrix4 Class Reference

this is a standard 4x4 translation matrix for objects. This is a standard 4x4 translation matrix for objects. It can be used for both 2D and 3D objects. By default it is a 3D matrix, It can be converted to a 2D matrix by using the function Set2D(). The matrix Layout is four columns, each one representing a different axis or translation.
