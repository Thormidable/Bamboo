cViewport Class Reference

cViewport allows a cCameras Render List to be rendered again from another position and rotation. cViewport are owned by cCamera and will render the cCamera s Render List again every frame. The cViewport s position and rotation can be set in exactly the same way as a camera. The area on the screen the cViewport renders to can be set by cViewportControl. The Position and perspective matrices used for rendering are set by cPerspectiveControl. When created, the cViewport will default to the first cCamera. It can be set to use a different cCamera s Renderlist on creation by passing it a pointer to the cCamera it should be a member of. More...

The cViewport s position and rotation can be set in exactly the same way as a camera. The area on the screen the cViewport renders to can be set by cViewportControl. The Position and perspective matrices used for rendering are set by cPerspectiveControl. When created, the cViewport will default to the first cCamera.
