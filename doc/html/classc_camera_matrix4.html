cCameraMatrix4 Class Reference

This is a translation matrix for a camera object A Special Matric for Cameras. All the translations are inverted. Distances are 'reversed', Local rotations are 'globalised' and Global roatations are 'localised'. Effectively all the translations are inverted before they are applied to this matrix. The matrix Layout is four columns, each one representing a different axis or translation. Xx, Yx, Zx, 0.0f, Xy Yy Zy 0.0f, Xz Yz Xz 0.0f, 0.0, 0.0, 0.0, 1.0f. The Position of the Camera Matrix is kept seperate to the 4x4 matrix. Functions which accept Camera Matrix objects or pointers automatically account for the differences in format.
