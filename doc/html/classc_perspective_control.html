cPerspectiveControl Class Reference

cPerspectiveControl affects how a cViewport or cCamera will render its view. This can be imagined as a lens. The user can set the field of view, both in width and height. You can set a near and far clipping plane. Adjustments to the parameters will only take effect after the function cPerspectiveControl::UpdateProjectionMatrix() is called. More...

Detailed Description

cPerspectiveControl affects how a cViewport or cCamera will render its view. This can be imagined as a lens. The user can set the field of view, both in width and height. You can set a near and far clipping plane. Adjustments to the parameters will only take effect after the function cPerspectiveControl::UpdateProjectionMatrix() is called.
