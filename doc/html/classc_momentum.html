cMomentum Class Reference

This Class will store an objects momentum (assumed mass of 1.0) and automatically update its matrix when the function Update() is called. This gives the user the options of global and local thrusts and will update the objects linear and angular momentum by the thrusts applied to it. An instance of cMomentum must be linked to a matrix. It will only update when the function Update() is called. More...

Detailed Description

This Class will store an objects momentum (assumed mass of 1.0) and automatically update its matrix when the function Update() is called. This gives the user the options of global and local thrusts and will update the objects linear and angular momentum by the thrusts applied to it. An instance of cMomentum must be linked to a matrix. It will only update when the function Update() is called.
