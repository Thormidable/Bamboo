cCollisionList Class Reference

This is generated by doing Collision Detection with an object. This will cache all the detected collisions with the object used for the collision detection. This allows the user to access the collisions in a different order to the order they were detected or to cache it for use later. The list is composed of a list of cCollisionListObject. More...

Detailed Description

This is generated by doing Collision Detection with an object. This will cache all the detected collisions with the object used for the collision detection. This allows the user to access the collisions in a different order to the order they were detected or to cache it for use later. The list is composed of a list of cCollisionListObject.
