Detailed Description

This class wil generate TBN Vectors for a mesh. Inherits from cAttributeData. This requires the mesh to have UV co-ordinates and Normals. TBN stands for Tangent, Binormal and Normal vectors. These are used in Normal Mapping shaders. This data will automatically be linked to Bb_Tangent,Bb_Binormal and Bb_Normal.
