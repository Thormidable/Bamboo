Detailed Description

template<class cX, class cS>
class cMainThread< cX, cS >

This Class is responsible for initialising and starting the Program. This is so the program can be created from a single line.

Template Parameters:
cXThis is the class that will first be called (The Core Process - it must inherit cProcess) and will initialise and start the game.
cSThis is the class that will be used for doing the Settings for the game and engine (it must inherit cUserSettings). This will create a window. Initialise the Engine and Create the first process - of type cX, before entering the main loop. This version of the class is for Windows. This is best called using the macro _START_PROGRAM(TYPE,SETTINGS).
