cShaderProgram Class Reference

A cShaderProgram() is a series of cShader() objects compiled into a program. cShaderProgram is a cFile() object. The cShaderProgram() finds a pointer to all the Shaders that compose it and compile them into a program. A cShader() object can be used in many different programs. The cShaderProgram() can be produced manually using the AttachShader() and Link() functions. The cShaderProgram() can be turned on with the use function. More...
