cTexture Class Reference

This class stores the data for rendering a 2D texture. This stores and processes the data for a 2D texture. This can be applied to objects to add maps to their surface. This can include Normal, specular, lighting or any other type of map. The data in the class can be modified and does update, but for textures which will receive a lot of updates, cDynamicTexture are faster. These inherit cFile and are loaded from IMFs. More...

Detailed Description

This class stores the data for rendering a 2D texture. This stores and processes the data for a 2D texture. This can be applied to objects to add maps to their surface. This can include Normal, specular, lighting or any other type of map. The data in the class can be modified and does update, but for textures which will receive a lot of updates, cDynamicTexture are faster. These inherit cFile and are loaded from IMFs.
